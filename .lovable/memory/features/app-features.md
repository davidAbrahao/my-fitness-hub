---
name: App features overview
description: All implemented features - cardio/movement blocks, PR evolution charts, exercise GIFs, calorie estimates
type: feature
---
- Cardio tab restored with 3 sub-tabs: Protocol, Registro, Blocos de Movimento
- Movement blocks: timer (30/45/60/90min) + checklist for quick exercises during work
- Walking registration: short 15min walks morning/afternoon
- PR evolution charts using recharts LineChart per exercise
- Exercise GIFs: external links to musclewiki.com for each exercise
- Calorie estimates: per workout (~280-550 kcal), per cardio activity, per movement block (~30 kcal)
- Diet page shows per-meal macro breakdown (P/C/G) and total calories
- BottomNav has 7 tabs: Painel, Treino, Dieta, Cardio, Check, Corpo, Tools
