# Project Memory

## Core
Projeto Barriga Zero — fitness tracking app, dark theme, neon green primary.
Mobile-first, Lovable Cloud (Supabase) com fila offline + cache localStorage. User: 33yo, 93.5kg, 170cm, 33% BF.
Training: 5 days/week ABCDE split. Diet target: 1800 kcal.
No light mode. Portuguese (PT-BR) UI.
PWA habilitado (vite-plugin-pwa) — SW NÃO registra em iframe/preview, apenas em produção publicada.
Insights IA via Lovable AI Gateway (google/gemini-2.5-flash) — server function `generateWeeklyInsights`.
